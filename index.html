Locked-In

This app is a password database. After creating a user account on our site you can then save your passwords in an encrypted database to access them from our chrome extension when logging into a new website I managed the entire back end of this project. I built user authentication and login persistance with Passport.js, Bcyrpt, and session cookies.

Traffic-Cameras

This application is a tracker for all of the red light and speed cameras in Chicago. You can populate a google map frame with all the cameras near an address or see the cameras you will pass along a google maps route. I mainly worked on the google maps api integration and calculations to populate a route with the cameras on the way.

Technologies Utilized:

Front End:
  • HTML
  • CSS
  • Bootstrap

Back End:
  • Firebase
  • google maps apis
  • data.cityofchicago.org
